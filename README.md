## Rune CLI

Simple CLI application to generate aseprite and .md files

### Usage

Edit the paths inside the _./configs/config.json_ and you are good to go!

### Commands

```
rune power "filename"
```

Create an aseprite file and launch it

```
rune glyph "filename"
```

Create an .md file and open it
